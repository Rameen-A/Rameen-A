# Feature: Project-details

## Goal

implement `projext-details.html`

## Designs

![target layout](./assets/images/spec.png)

## Work
- Create a detailed page for each project
- Display the project title in middle 
- Show a screenshot or preview image of the project
- Include a description explaining:
  - what the project does
  - what technologies were used
- Add a section explaining technical challenges faced during development
- Add a section showing what I learned from the project
- Buttons should change color on hover
- Images should resize properly on smaller screens
- Text should remain easy to read on desktop and mobile devices
- Images should resize correctly on smaller screens


## Deliverables
- Project detail page loads correctly
- "Go back on top" button
- Hover effects work on buttons
- Content is organized clearly
- Layout is responsive on desktop and mobile
- Layout remains responsive on smaller screens
- No overlapping text or broken spacing
- HTML uses semantic structure for accessibility
